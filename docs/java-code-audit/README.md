# ☕ Java 安全审计探索
> 记录从零开始学习 Java 代码审计的点点滴滴。

### 📌 学习初衷
作为网安专业学生，深知 Java 在企业级应用中的核心地位。本模块主要记录我对 Java 基础安全机制的理解以及常见漏洞链的分析，目前尚处于“打基础”阶段。

### ✍️ 核心笔记内容
* **基础储备**: 反射机制、类加载器（ClassLoader）、RMI/JNDI 基础概念。
* **OWASP Top 10**: 重点记录 SQL 注入、SSRF、目录遍历在 Java 端的表现与修复。
* **框架安全**: 尝试分析 Spring、Struts2 等框架的简单历史漏洞。
* **审计小结**: 记录使用 IDEA 进行动态调试、利用漏洞扫描工具辅助人工审计的心得。
