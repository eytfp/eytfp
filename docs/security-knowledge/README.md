# 🛡️ 网安基础知识沉淀

> **「 🚩 循序渐进，不积小流无以成江海 」** > 这里是我在专业课学习和技术积累过程中，对计算机基础与网络协议的观察笔记。

### 📌 模块简介

这个模块主要用于记录那些“看不见”但又无处不在的基础知识。安全技术虽然千变万化，但底层的 **TCP/IP 协议**和**操作系统逻辑**是相对稳定的。我希望通过整理这些笔记，把“内功”练好，为以后深挖 Web 安全打下坚实的基础。

------

### 🔍 知识图谱 · 学习索引

#### 🌐 1. 网络协议基础 (Traffic Analysis)

- **📡 协议运行逻辑**: 观察并记录 `HTTP/HTTPS`、`DNS`、`DHCP`、`ARP` 等协议在真实环境下的交互流程。
- **🔍 数据包深度观察**: 练习使用 **Wireshark** 或 **Tcpdump** 抓包，分析 TCP 三次握手、四次挥手的状态变迁。
- **🏢 内网通信初探**: 了解局域网（LAN）环境，学习什么是 `VLAN 隔离` 以及基础的路由交换概念。

#### 🐧 2. 操作系统安全 (System Security)

- **⌨️ Linux 命令行**: 整理常用的 Linux 操作命令、用户权限模型（`u/g/o`）以及简单的 Shell 自动化小脚本。
- **💻 Windows 机制**: 记录 Windows 用户组权限、基本的安全策略配置（如组策略、注册表基础）。
- **📦 容器化基础**: 简单记录 Docker 的安装配置、镜像管理以及容器的基础隔离概念。

#### 🛠️ 3. 基础环境与标准 (Standards)

- **⚙️ 实验环境搭建**: 记录如何快速配置镜像源、搭建各类 Web 漏洞实验环境（如 `VULHUB`）。
- **📖 行业术语理解**: 整理对 `CVE` 编号、`CWE` 分类以及漏洞危险等级评定标准的个人学习笔记。
- **📜 法律法规学习**: 简单了解《网络安全法》中与技术人员相关的基础常识，确保技术研究合法合规。
