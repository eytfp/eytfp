# 🏰 内网渗透基础学习
> 探索边界之内的世界，学习域环境下的攻防博弈。

### 📖 学习背景
内网渗透是安全能力的分水岭。本模块记录我作为新手，在模拟环境（如靶场）中学习内网基础知识的总结。

### 🧭 学习路线
* **环境搭建**: 记录如何利用虚拟机搭建简单的单域环境及多级网络。
* **横向移动基础**: 学习 `net` 命令、PSEXEC 等基础横向工具的使用原理。
* **隧道搭建**: 总结 FRP、NPS 等工具在内网穿透中的应用场景。
* **身份认证**: 尝试理解 Kerberos 协议、NTLM 认证的基础流程。

### ⚠️ 谦虚声明
内网知识体系极其庞大，目前的笔记仅局限于最基础的入门操作，深感路漫漫其修远兮。
