# 🛠️ Go 语言安全工具开发
> “工欲善其事，必先利其器”，尝试用 Go 语言实现自己的安全小工具。

### 🏗️ 为什么选择 Go？
Go 语言原生的高并发特性和简洁的语法非常适合编写高性能的扫描器和网络工具。本模块记录我从零开始学习 Go 语法并将其应用于安全开发的实验过程。

### 🛠️ 目前的“小轮子”
* **Port Scanner**: 基于 `net` 包实现的简单端口扫描，正在优化 SYN 扫描效率。
* **Info Leak Hunter**: 用于自动检测网页源代码中敏感信息的正则匹配工具。
* **Web Fuzzer**: 练习编写多线程的参数探测小脚本。

### 📚 学习目标
不追求复杂的架构，重点在于理解网络协议栈（如 TCP/IP）在代码层面的实现，提升代码审计时的敏锐度。
