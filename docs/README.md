# 安全学习笔记

面向 **求职展示** 的极简技术博客，内容由 Markdown 编写，托管于 **GitHub Pages**，无服务器、无备案成本。

## 站点能力

| 能力 | 说明 |
| --- | --- |
| 写作 | 任意编辑器或 GitHub 网页直接粘贴 Markdown |
| 代码 | Prism 高亮 + 行号（见各示例文） |
| 分类 | 五大目录对应五大方向；侧边栏即目录 |
| 标签 | 每文顶部 `标签` 行，站内搜索可搜到正文 |
| 搜索 | 左上角搜索框，全文索引 |
| 项目 | 见顶部导航 **项目与链接** |

## 分类说明

1. **Java 代码审计** — `java-code-audit/`
2. **SRC** — `src/`
3. **内网** — `intranet/`
4. **网安知识** — `security-knowledge/`
5. **CTF** — `ctf/`

新建文章：复制对应目录下的 `_template.md`，改名后在 `_sidebar.md` 里加一条链接。详细步骤见 [维护说明：增删改查](guide/edit-guide.md)。
